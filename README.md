Данная программа используется для редактирования, удаления, добавления пользователей и вывода их списка. Имеется две роли авторизации ; "Администратор" и "Пользователь". Чтобы запустить это программу необходимо перейти в следующую папку на диске д: .D:\vs\BurgerPlus\bin\Debug. В этой папке необходимо открыть файл с расширением exe с наименованием - "BurgerPlus.exe". Версия программы - VisualStudio 2022/
